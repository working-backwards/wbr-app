# Publishing & Export

## Publish to a URL

1. Generate a report
2. Click the green **PUBLISH** button
3. Optionally check "Require a password" and enter a password
4. Click **PUBLISH**
5. Copy the generated URL

## JSON export

Click the green **JSON** button to download the deck as a JSON file.
Import into Tableau, Looker, or other visualization tools.

## Cloud storage

Published reports can be stored on S3, GCP, or Azure. Set via
environment variables (see README for configuration).
